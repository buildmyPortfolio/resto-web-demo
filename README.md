# Fino Deli – Restaurant Website

A fancy, fully responsive restaurant website for **Fino Deli** — *The First Wine Bar & Delicatessen in Marikina*.

## 🌐 Live Site
Hosted on GitHub Pages: `https://<your-username>.github.io/<repo-name>/`

## 📁 Project Structure
```
├── index.html          # Main page
├── styles.css          # All styles
├── script.js           # Interactivity (tabs, lightbox, navbar)
├── Logo.jpg            # Restaurant logo
├── Menu/               # Dish photos (.webp / .jpg)
└── Menu List/          # Printed menu images (.jpg)
```

## ✨ Features
- Fixed glassmorphism navbar with scroll effect
- Full-screen hero with parallax overlay
- Animated marquee ribbon
- About section with story and service highlights
- Interactive tabbed menu (12 categories, full menu)
- Masonry-style photo gallery with lightbox viewer
- Google Maps embed + contact info
- Reservations call-to-action section
- Fully responsive (mobile, tablet, desktop)

## 🚀 Deploy to GitHub Pages

1. Create a new repository on GitHub
2. Push all files to the `main` branch
3. Go to **Settings → Pages → Source → Deploy from branch → main / root**
4. Your site will be live at `https://<username>.github.io/<repo>/`
